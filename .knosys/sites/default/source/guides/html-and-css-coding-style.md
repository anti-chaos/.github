## 命名

### HTML/CSS class

页面结构及样式中所使用的 class 用继承并扩展了「[BEM](http://getbem.com/)」命名法的 [SUIT CSS 命名方式](https://github.com/suitcss/suit/blob/master/doc/naming-conventions.md)。

## 其他

### HTML

严格遵守「分离原则」，不在页面中编写内联样式和非配置类的内联脚本。
